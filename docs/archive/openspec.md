# Archivo de OpenSpec

`openspec/` ya no es fuente viva de verdad para el proyecto.

## Regla actual

- `sdd/` contiene el roadmap vivo, specs, diseño y task plans
- `docs/` contiene setup, operación y referencia técnica
- `openspec/` queda como histórico transitorio mientras su contenido útil se migra o se archiva

## Uso esperado

Consultalo solo para contexto histórico. Si un documento contradice al código o a `sdd/`, prevalecen el código y `sdd/`.
