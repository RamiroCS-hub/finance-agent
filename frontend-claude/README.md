# Frontend Claude

Landing estatica para Tesorero con CTA directa al bot de Telegram `@AnotameloBot`, sin backend propio.

## Deploy en Vercel

Configura el proyecto con:

- Root Directory: `frontend-claude`
- Framework Preset: `Other`
- Build Command: vacio

## Como funciona

Los CTA principales abren `https://t.me/AnotameloBot` en una nueva pestaĂąa para iniciar la conversacion desde Telegram.

## Uso

- Landing publica: `index.html`
- Admin informativo: `admin.html`

No hay backend ni base de datos. Si queres persistencia real despues, conviene volver a Sheets o a una base.
